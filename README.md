# Fraud Detection System

This project is about detecting fraudulent transactions using machine learning.

## What I did
- Loaded and explored the dataset using Pandas
- Cleaned the data and removed unnecessary columns
- Found that the dataset was highly imbalanced (very few fraud cases)
- Used SMOTE to balance the dataset
- Trained models like Logistic Regression and Random Forest
- Evaluated the model using precision, recall, and F1-score

## Tools I used
- Python
- Pandas, NumPy
- Scikit-learn
- imbalanced-learn (SMOTE)

## What I learned
- How to handle imbalanced datasets
- Why accuracy is not always a good metric
- How different models perform on real-world data

## Dataset
https://www.kaggle.com/mlg-ulb/creditcardfraud
